# Foxy Desktop Companion

Foxy is a little 2D pixel Companion that walks around your screen to accompany while you're working, gaming, etc.

## Features

- Follows cursor
- random wandering toggeable

## Installation
1. Go to [Releases](https://github.com/glitchchh/foxy/releases)
2. Download `foxy.exe`
3. Run it. (Note: You may need to click "More Info" -> "Run Anyway" on Windows)
